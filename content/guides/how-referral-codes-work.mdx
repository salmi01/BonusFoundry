Referral codes are invitations. A provider uses the code or referral link to connect a new user to the person who shared it, then checks whether the account and first transfer meet the active terms.

## Direct answer

A referral code may unlock a reward, but only if the provider's current terms allow it. Common requirements include being a new customer, signing up through the correct link, completing verification, and making a qualifying first transfer.

## What to check first

- Whether the offer is available in your country.
- Whether the destination country or corridor is eligible.
- The minimum transfer amount.
- The deadline for completing the qualifying action.
- Whether the reward appears before or after the first transfer.

## Practical steps

1. Choose the provider you want to use.
2. Open the referral link or copy the code before signup.
3. Check the offer terms in the provider app.
4. Complete verification if required.
5. Send a qualifying first transfer and keep the receipt.

## Mistakes to avoid

- Treating a referral code as official provider support.
- Assuming a code works in every country.
- Creating an account before applying the referral link.
- Expecting a reward before the qualifying transfer clears.

## FAQ

### Can I use a referral code after signing up?

Usually no. Many providers require the code or link before account creation.

### Do referral codes expire?

They can. Provider campaigns may change by country, route, or date.
