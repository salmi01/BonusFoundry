Referral codes connect a new user to an invitation and may unlock a reward when the provider's conditions are met. The important checks are when to apply the code, who is eligible, what transfer qualifies, and when the provider applies the reward.

## Direct answer

A referral code usually works by linking a new account or first transfer to an existing user's invitation. Common requirements include being a new customer, applying the code before signup or the first transfer, completing verification, and making a qualifying transfer.

## How referral codes work

1. A user receives a referral code or referral link.
2. The new user applies the code or opens the link before signup or the first qualifying transfer.
3. The provider checks country, account, amount, payment method, destination, and verification requirements.
4. The provider applies the reward, fee discount, credit, or bonus only if the current terms are met.

## Practical steps

1. Choose the provider you want to use.
2. Open the referral link or copy the code before signup or checkout.
3. Check sender country, destination country, minimum transfer, expiry, and payout timing.
4. Complete identity or payment verification if required.
5. Send a qualifying first transfer only after the offer appears or the provider accepts the code.
6. Keep the offer screen and transfer receipt until the reward is resolved.

## Checklist before using a referral code

- Code or link is applied before account creation or first transfer.
- Sender country and recipient country match the offer.
- Transfer amount and delivery method match the terms.
- Verification is complete or in progress.
- Reward timing and expiry are understood.
- Screenshots or receipts are saved.

## Common mistakes

- Assuming a code works in every country or corridor.
- Creating an account before applying the referral link or code.
- Expecting a reward before the qualifying transfer clears.
- Ignoring payment-method, destination, or minimum-transfer rules.

## FAQ

### Can I use a referral code after signing up?

Usually no. Many providers require the code or link before account creation or the first qualifying transfer. Check the provider's support page if the offer was visible but did not attach.

### Do referral codes expire?

Yes. Referral campaigns can expire or change by country, route, transfer amount, or date.

### Can existing users use referral codes?

Usually referral codes are for new users, but some providers may run offers for existing accounts. The provider's live offer terms decide eligibility.

### What should I do if the reward is missing?

Check whether the code was applied, the transfer qualified, verification is complete, and the reward timing has passed. Contact provider support with the offer screen and transfer receipt.
