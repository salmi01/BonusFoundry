Promo codes and referral codes can both help eligible users access a discount, reward, or transfer bonus. The main difference is where the offer comes from and which conditions the provider applies.

## Direct answer

A promo code is usually part of a provider campaign. A referral code is usually tied to an invitation from an existing user. Either type can have country limits, account-status rules, expiry dates, payment-method rules, and minimum transfer requirements.

## When a referral code matters

Referral codes matter most when a provider rewards new users for joining through an invitation. The provider may also reward the person who shared the code.

## When a promo code matters

Promo codes are more often used for seasonal campaigns, first-transfer discounts, or targeted offers. They may not be connected to a specific user.

## Practical steps

1. Identify whether the provider calls the offer a referral, promo, invitation, or welcome offer.
2. Check who is eligible: new users, existing users, inviters, or recipients.
3. Confirm whether the offer stacks with other discounts.
4. Save the terms before completing a transfer.
5. Confirm the code is accepted before sending money.

## Checklist before using either code

- Offer type is clear: referral, promo, welcome, cashback, fee discount, or transfer credit.
- Sender country and destination are eligible.
- Transfer amount, payment method, and delivery method match the terms.
- Expiry date and reward timing are clear.
- The provider accepts the code or link before checkout.

## Common mistakes

- Assuming referral and promo codes can be combined.
- Entering a promo code after the first transfer.
- Ignoring route and country restrictions.
- Assuming a referral reward and fee discount have the same timing.

## Before using either code

Read the current terms in the app or provider website. If the terms shown by the provider differ from a guide, the provider's terms control eligibility.

## FAQ

### Which code should I use?

Use the offer that the provider confirms is valid for your account and transfer route.

### Can both codes expire?

Yes. Referral and promo offers can both expire or change.

### Can I use a promo code and referral code together?

Sometimes, but do not assume stacking is allowed. Check whether the provider permits more than one offer on the same account or transfer.

### Which code is better?

The better code is the one that produces the best total result after fees, exchange rate, delivery method, reward timing, and eligibility rules are considered.
