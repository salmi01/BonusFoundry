# Goal

Create dynamic provider pages.

# Read First

- DATA_MODEL.md
- ROUTING.md

# Requirements

Dynamic route:

```text
/providers/[slug]
```

Each page should contain:

- Overview
- Welcome bonus
- Eligibility
- Supported countries
- How it works
- FAQ
- Related pages
- Referral section
- Disclosure

Use static data.

# Deliverables

Provider page template.

# Acceptance Criteria

Adding one provider in data should automatically create a page.

# Out of Scope

Blog.
Homepage.
