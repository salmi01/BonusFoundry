# Goal

Initialize the project using the architecture defined in `/docs`.

# Read First

- PROJECT_CONTEXT.md
- ARCHITECTURE.md

# Requirements

- Next.js 15
- TypeScript
- Tailwind CSS
- App Router
- MDX support
- ESLint
- Prettier
- shadcn/ui
- No database
- No authentication

Create the following folders:

```text
app/
components/
content/
data/
lib/
public/
docs/
```

Configure:

- Metadata
- Fonts
- Global layout
- Theme
- Path aliases

# Deliverables

A clean project ready for feature development.

# Acceptance Criteria

- npm install works
- npm run dev works
- npm run build succeeds
- No TypeScript errors
- No unused dependencies

# Out of Scope

Pages.
SEO.
Business logic.
