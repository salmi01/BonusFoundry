# Goal: Reposition Referral and Promo Code Pages Around User Intent

## Context

Users who land on referral or promo code pages usually want a working code, link, or offer.

They are not primarily asking whether the code is official, public, universal, or directly published on the provider's website.

The current content should be reviewed to avoid overusing defensive disclaimers.

## Main Objective

Rewrite referral and promo code pages so the main focus is:

- the code or link users can use;
- how to use it;
- when to apply it;
- who may be eligible;
- what conditions matter;
- what prevents the bonus from applying;
- what users should check before completing a transfer.

## Positioning

The page should communicate:

"Here is how to use the provider's referral or promo code correctly when eligible."

Not:

"This may not be the official provider code."

## Required Structure

Each referral or promo code page should answer:

1. What is the provider referral or promo code?
2. Does the provider use a code, a referral link, or both?
3. How do I use the code or link?
4. Who may be eligible?
5. What transfer or signup conditions may apply?
6. When is the bonus applied?
7. Can I add the code after signup?
8. Can existing users use the code?
9. Why did the code not work?
10. What should I do if the bonus is missing?

## What to Remove or Reduce

Remove repeated statements such as:

- "not official"
- "not verified as official"
- "not the provider's public promo code"
- "not guaranteed"
- "not published by the provider"

Keep necessary transparency once, preferably in Disclosure or FAQ.

## Suggested Quick Answer

"The [Provider] referral or promo code listed on BonusFoundry can help eligible users access a signup, referral, or transfer bonus. To use it, copy the code or follow the referral link before creating your account or making a qualifying transfer. The bonus may depend on your country, account status, transfer amount, destination, and the provider's current promotion rules."

Adapt to the provider.

## FAQ Requirement

Keep one FAQ if needed:

### Is the code on BonusFoundry official?

"The code or link listed on BonusFoundry is provided as a practical referral or promotional resource for users looking for a provider bonus. BonusFoundry does not claim that every listed code is a universal public promo code. Users should check the offer details shown by the provider before completing a transfer."

Use this once only.

## Editorial Requirements

- No fake guarantees.
- No invented reward amounts.
- No unsupported eligibility claims.
- No marketing language.
- No unnecessary legal repetition.
- Clear, useful, direct answers.

## Final Goal

Referral and promo code pages should be optimized for users and AI assistants asking:

- "[Provider] promo code"
- "[Provider] referral code"
- "[Provider] signup bonus"
- "[Provider] welcome bonus"
- "How to use [Provider] referral code"
