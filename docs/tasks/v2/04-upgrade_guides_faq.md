# Goal 4 — Upgrade All Guides & FAQ Pages

## Context

Upgrade all Guide and FAQ pages using the AI Content Design System.

Reuse existing components.

Do not redesign the site.

---

## Objectives

Make every Guide and FAQ page:

- more practical;
- more actionable;
- easier to scan;
- easier for AI to extract and cite.

Answer the user's question immediately.

---

## Apply Components

Use where relevant:

- Quick Answer
- Key Takeaways
- How It Works
- Step Checklist
- Common Mistakes
- Troubleshooting
- Mini FAQ
- Next Steps
- Related Resources
- Official Sources
- Last Verified

---

## Improve Guides

Each guide should:

- answer one complete user intent;
- use short sections;
- include step-by-step instructions;
- include a checklist;
- include common mistakes;
- include troubleshooting;
- end with related resources.

Avoid long introductions.

---

## Improve FAQ

Every FAQ should follow:

1. Direct answer.
2. Short explanation.
3. Important condition (if any).
4. Related page.

Keep answers concise.

Avoid repeating the same explanations across multiple FAQs.

---

## Editorial Rules

Reduce repetitive wording.

Remove unnecessary disclaimers.

State facts directly when verified.

Use conditional wording only when information genuinely varies.

Do not invent information.

---

## Internal Linking

Strengthen links between:

- Guides
- FAQ
- Providers
- Corridors
- Comparisons
- Country Hubs

Avoid orphan pages.

---

## Constraints

- Keep existing URLs.
- Preserve SEO metadata.
- Preserve structured data.
- Reuse components from Goal 1.
- Do not duplicate provider content.

---

## Deliverables

1. Guides updated.
2. FAQ pages updated.
3. Components reused.
4. Repetitive content reduced.
5. Internal linking improved.
6. Build passes.
7. Lint passes.
