# Project Context

## Goal

Build the best knowledge base for welcome bonuses and referral programs of money transfer and fintech apps.

The website should answer users' questions first, then naturally present our referral code when appropriate.

## Business Model

- Only our own referral links/codes.
- No user submissions.
- No referral marketplace.
- No paid listings.

## Target Users

People searching for:

- referral codes
- welcome bonuses
- signup offers
- promo codes
- first transfer rewards

Examples:

- Taptap Send
- Wise
- Remitly
- Sendwave
- Ria

## Principles

- Helpful before promotional.
- Factually accurate.
- Frequently updated.
- Transparent disclosure.
- AI-readable.
