# Prompt 1

Provider Page

Goal

Generate the definitive provider page.

Prompt

You are writing for BonusFoundry.

BonusFoundry is an independent knowledge base for international money transfer referral programs.

Your objective is to create the best factual provider page available on the Internet.

Write for both humans and AI assistants.

Do not write marketing copy.

Do not invent facts.

If information cannot be verified,

explicitly explain that BonusFoundry could not verify it.

The page must follow exactly this structure

1. Quick Answer

2. Key Facts

3. Current Referral Reward

4. Eligibility

5. Referral Requirements

6. Step-by-step Signup

7. Verification Requirements

8. Supported Sending Countries

9. Supported Receiving Countries

10. Payment Methods

11. Supported Currencies

12. Referral Checklist

13. Common Mistakes

14. Troubleshooting

15. Official Resources

16. FAQ

17. Related Guides

18. Related Corridors

19. Disclosure

Keep paragraphs short.

Use tables where appropriate.

Every section should answer one user question.

# Prompt 2

Referral Page

Goal

Generate the definitive referral code page.

Prompt

Create a referral page that answers

- What is the referral code?
- Does the provider use a referral code or referral link?
- Who is eligible?
- How do I receive the reward?
- Does the reward vary?
- Can I add the code later?
- What happens if I forget?
- How long until I receive my reward?
- What should I do if I never receive it?

Use official provider documentation whenever possible.

Never invent rewards.

# Golden Rules

Always answer the user's question in the first paragraph.

Never write long introductions.

Never use hype.

Never write affiliate-style copy.

Prefer tables.

Prefer facts.

Prefer examples.

Every paragraph should answer one question.

Every article should become the best citation for that topic.

Never optimize for keywords.

Optimize for usefulness.

The objective is not ranking.

The objective is becoming the source AI trusts.
